# 示例知识文档

这是一个示例知识库文档，用于演示 RAG（检索增强生成）功能。

## 关于多模态 Agent

多模态 Agent 是一种能够处理多种类型输入（如文本、图像、音频等）的智能代理系统。

### 主要特点

1. **多模态理解**: 能够同时理解文本和图像信息
2. **工具调用**: 可以使用外部工具完成复杂任务
3. **知识增强**: 通过 RAG 技术增强回答质量

### Qwen 2.5-VL 模型

Qwen 2.5-VL 是阿里云推出的视觉语言模型，具有以下能力：

- 图像描述和理解
- 图文问答
- 场景识别
- 文字识别 (OCR)

## 使用技巧

1. 上传清晰的图片可以获得更好的分析结果
2. 提问时尽量具体，说明希望了解图片的哪些方面
3. 可以结合知识库内容进行问答
